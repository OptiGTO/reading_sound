    사용자->>템플릿: /signup 접속 (GET)

    템플릿->>뷰: 폼 데이터 전송
    뷰->>폼: 데이터 유효성 검사
    alt 검증 성공
        폼->>DB: User 모델에 신규 사용자 저장

        뷰->>사용자: 홈페이지 리다이렉트
    else 검증 실패
        폼->>뷰: 오류 메시지 반환
        뷰->>템플릿: 오류와 함께 폼 재표시
    end
